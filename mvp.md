# MVP

## MVP name

**YouTube Niche Analyzer + Trend Radar**

## MVP goal

Help a user identify:
- promising niches
- emerging topic angles
- rising trends
- competitor patterns
- content opportunities worth testing now

## What MVP must do

### 1. Niche input
User enters:
- keyword
- topic
- category
- region
- Shorts / long-form / both

### 2. Related niche expansion
System returns:
- related topics
- related queries
- adjacent niche angles
- fresh variants

### 3. Video and channel scan
System pulls:
- recent videos
- channel names
- view counts
- publish timing
- simple engagement indicators

### 4. Transcript enrichment
System:
- pulls transcripts where available
- extracts repeated content angles
- identifies repeated hooks and themes

### 5. Competitor snapshot
System shows:
- most active relevant channels
- repeated successful formats
- common topic angles
- oversaturated patterns

### 6. Niche scoring
System outputs:
- opportunity score
- freshness score
- competition score
- format fit
- short explanation

### 7. AI summary + ideas
System generates:
- why this niche matters now
- risks
- best entry angle
- 10–20 video ideas

### 8. Watchlist
User can save:
- niche
- competitor channel
- keyword cluster

## What MVP should NOT include yet

- browser extension
- full vidIQ-style overlay
- advanced SEO scoring
- thumbnail grading
- upload assistant
- team collaboration
- huge automation suites

## Suggested MVP score model

Score out of 100:

- Demand: 20
- Freshness: 20
- Competition: 15
- Small-channel breakout signal: 15
- Repeatability: 10
- Monetization: 10
- Shorts fit: 5
- Long-form fit: 5

## MVP outputs

A result screen should show:

- niche name
- total score
- why it is rising
- how crowded it is
- key competitors
- repeated angles
- best entry strategy
- content ideas

## MVP success condition

A user can type a niche and leave with:

- a ranked opportunity score
- a reasoned explanation
- clear competitor context
- content ideas worth testing
